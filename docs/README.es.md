**JoséBD** (también conocido como **ZéBD**) es una base de datos NoSQL ligera y orientada a documentos que almacena datos de forma nativa en archivos JSON. Desarrollada en Brasil, fue diseñada para ofrecer simplicidad y eficiencia en aplicaciones de pequeña escala, lo que la hace perfecta para servidores simulados, prototipado local o implementaciones rápidas con API RESTful, GraphQL y tRPC.
